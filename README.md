# Davy Jones' Locker
This is a dead-man's switch contract where you can:
- Create Chest
- Add Treasure
- Delay Unlock
- Unlock Chest

## Create Chest
Deposit initial ADA

## Add Treasure
Deposit more ADA to the chest

## Delay Unlock
Postpone chest unlocking by 30 days

## Unlock Chest
Redeem all ADA from the chest
